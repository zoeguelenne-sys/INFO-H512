# INFO-H512
Repository containing the code of the INFO-H512 project and clear instructions for reproducing results. 

Set up: 

## Author's library 
conda create n- INFOH512 python=3.10 -y
conda activate INFOH512
pip install git+https://github.com/jjbrophy47/tree_influence.git

## Basic libraries 
pip install lightgbm 
pip install pandas 
pip install matplotlib
pip install numpy
pip install scipy
pip install scikit-learn

## Others 
pip install ipykernel

## SHAP API 
pip install shap
